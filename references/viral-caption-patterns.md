## 套路 1：emoji 密度高（每段 3-8 个）

| 槽位 | emoji 用途 | 例子 |
|---|---|---|
| 标题 | 视觉钩子 | 周末｜不去会后悔的 🏞️ |
| 段落首 | 视觉断点 | 🌸 必冲 / 📍 地点 / 🍢 食物 |
| 关键词 | 强化情绪 | 超～宝藏 ✨ / 心瞬间安静 🏔️ |
| 结尾 | 收尾标记 | 💕 / 📌 / 👇 |

**反例**：`这个周末我去了一个地方，那里很漂亮` → 没 emoji = 不像小红书

## 套路 2：标题党 + 反差钩子

| 类型 | 模板 | 例子 |
|---|---|---|
| 反差 | 「免费 + 不去后悔」 | 川西最美的地方都是**免费**的... |
| 数字 | 「3 个 / 5 个 / 7 个」 | 成都 5 个小众地本地人才知道 |
| 避坑 | 「别再去 X 了」 | 别再去宽窄巷子了！ |
| 拟人 | 「XX 会生气的」 | 这个周末不去这里，夏天会生气 |

## 套路 3：必冲打卡点分段

```
🌟必冲打卡点
📍 四姑娘山·双桥沟漂流
夏日限定快乐！缓缓漂过高山草甸...

📍 甲根坝
闯入俄色花海和青稞田园，像走进新海诚动画！
```

**规律**：
- emoji + 地理名 + 类型
- 一句话情绪 + 一个细节画面
- 5-7 个地点最常见

## 套路 4：Day-by-Day 行程

```
🗺️5天不累行程
Day1：全国→成都集合
晚上直奔奎星楼街吃串串...

Day2：成都→四姑娘山·双桥沟→丹巴
上午漂流玩水拍大片...

Day3：丹巴→墨石公园→姑弄村→八郎生都
穿越异域星球墨石...
```

**关键**：
- 必须有 Day 1（起点+集合）
- 中间 Day 有"上午/下午/晚上"分段
- 末尾留 1 天"返程"
- 配"住宿+食物+活动"三件套

## 套路 5：食物穿插

| 场景 | 食物用法 |
|---|---|
| 集合点 | "晚上直奔 X 街吃串串，感受巴适夜生活" |
| 路上补给 | "中途在 X 镇吃一碗 X" |
| 拍照道具 | "冰粉碗记得带回家当道具" |
| 价格锚点 | "30 块就够了" / "人均 50" |

**核心**：食物 = 烟火气 = 真实感 = 小红书灵魂

## 写作 SOP（5 步）

| 步 | 做什么 |
|---|---|
| 1 | 选模板（治愈/美食/小众） |
| 2 | 填具体地点（成都优先用：锦江/人民公园/宽窄巷子/东郊记忆/玉林/奎星楼） |
| 3 | 加 emoji（每段 3-5 个） |
| 4 | 加 1 句金句（结尾或 Day 后） |
| 5 | 配 3-5 张图（实拍+角度变化） |

## 关键红线

| ❌ 不要 | ✅ 要 |
|---|---|
| 长段落无 emoji | 每段 3-5 个 |
| "我推荐 XX" | "必冲 / 挖到宝藏 / 偷偷告诉你们" |
| 全是形容词 | 数字+价格+具体地名 |
| 没有食物 | 食物穿插是灵魂 |
| 没有 Day-by-Day | 行程化（哪怕 1 天也分段） |
